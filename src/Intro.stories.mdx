# Get started

Trabee Design System is a reusable component library that helps Trabee contributors build UIs faster. The goal is to make building durable UIs more productive and satisfying.

## Install

Trabee components are written in React, and its stories are written in Component Story Format. It requires Storybook version 6.1.20 and up.

Add SDS to your project.

```
npm install --save @trabee/ui
```

## Use

Import components you want into your UI

```tsx
import { MotionButton } from "@trabee/ui";
```

and use them like so

```tsx
const Example = () => (
  <MotionButton text="Hello Trabee" onClick={() => console.log("Hello Trabee")} />
);
```
