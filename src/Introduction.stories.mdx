# Get started

Storybook Design System is a reusable component library that helps Storybook contributors build UIs faster. The goal is to make building durable UIs more productive and satisfying.

## Install

SDS components are written in React, and its stories are written in Component Story Format. It requires Storybook version 5.2-beta and up.

Add SDS to your project.

```
npm install --save @trabee/ui
```

## Use

Import components you want into your UI

```
import { Button } from ‘@trabee/ui’;
```

and use them like so

```
const example = () => (
  <div>
    <Button text="Hello Trabee" onClick={() => console.log("Hello Trabee")} />
  </div>
)
```
